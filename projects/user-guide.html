01 Folder Structure
MLJ/projects/ ├── index.html ← Dashboard (main entry) ├── user-guide.html ← यो file │ ├── web-engine/ │ ├── css/ │ │ └── global.css ← design system (सबैले use गर्छन्) │ └── js/ │ ├── dashboard.js ← data, state, theme, navigation │ └── router.js ← page rendering │ ├── contents/ │ ├── lab-template.html ← नयाँ lab बनाउन copy गर्ने │ ├── java/ │ │ ├── index.json ← Java topics data │ │ └── memory-lab.html ← OOP Memory Lab │ ├── js/ │ │ ├── index.json │ │ └── *.html ← future JS labs │ ├── cpp/ │ │ └── index.json │ └── dsa/ │ ├── index.json │ └── dsa-lab.html ← DSA Memory Lab │ └── assets/ └── image/ ├── java/ ← .svg drawio files ├── js/ └── dsa/
Golden Rule: dashboard.js र router.js — यी दुइटा छुनु नपरोस् जतिसक्दो। नयाँ content थप्न index.json मात्र edit गर्नुस्। Lab बनाउन lab-template.html copy गर्नुस्।
02 नयाँ Topic थप्ने
सबैभन्दा common task। Generics सिक्यो, topic थप्नु छ — index.json मात्र edit गर्नुस्।
1
Language को index.json खोल्नुस्
उदाहरण: Java topic → contents/java/index.json
2
topics array मा नयाँ object थप्नुस्:
{ "id": "j19", // unique ID — j=java, js=javascript, d=dsa "name": "Generics", // topic name "status": "learning", // "todo" | "learning" | "done" "lab": "generics.html", // lab file (null if no lab yet) "labSection": "intro", // tab id in lab (optional) "notes": ["../../assets/image/java/generics_notes.svg"], "tags": ["generics", "type-safety", "bounded"] }
3
File save गर्नुस् → Browser refresh गर्नुस्।
Dashboard मा automatically देखिन्छ। Badge count update हुन्छ। Progress tracker मा reflect हुन्छ।
Status values: "todo" — सुरु गरेको छैन · "learning" — अहिले सिक्दैछ · "done" — complete
Dashboard मा dropdown बाट पनि change गर्न मिल्छ — localStorage मा save हुन्छ।
03 नयाँ Language थप्ने (e.g. Python)
1
Folder र JSON बनाउनुस्:
mkdir contents/python touch contents/python/index.json
2
contents/python/index.json fill गर्नुस्:
{ "language": "python", "color": "#3776ab", "icon": "🐍", "topics": [ { "id":"py1", "name":"Basics", "status":"todo", "lab":null, "notes":[], "tags":["python"] } ] }
3
web-engine/js/dashboard.js मा langs object मा थप्नुस्:
langs: { java: { ... }, js: { ... }, python: { label:'Python', icon:'🐍', color:'#3776ab', jsonPath:'./contents/python/index.json' }, }
4
index.html Sidebar मा nav button थप्नुस्:
<!-- Languages section भित्र --> <button class="nav-item" data-page="python" onclick="MLJ.navigate('python')"> <span class="nav-icon" style="background:rgba(55,118,171,.1);color:#3776ab">🐍</span> Python <span class="nav-badge" id="badge-python" ...>—</span> </button>
04 नयाँ Lab बनाउने
lab-template.html copy गर्नुस् — सबै boilerplate already छ।
1
Template copy गर्नुस्:
cp contents/lab-template.html contents/java/generics.html
2
File खोलेर यी ४ वटा चीज change गर्नुस्:
<!-- 1. Title --> <title>Generics Lab — MLJ</title> <!-- 2. Language color --> <body style="--lang-color: #f89820"> <!-- 3. Lab header --> <div class="lab-title">☕ Generics</div> <div class="lab-sub">// java · type safety · MLJ</div> <!-- 4. Tab names --> <button class="lab-tab active" onclick="switchTab('intro')">Intro</button> <button class="lab-tab" onclick="switchTab('bounded')">Bounded Types</button>
3
Steps array fill गर्नुस् (animation को लागि):
const s1Steps = [ { info: '<strong>Step 1:</strong> Generic type T भनेको placeholder हो।', stack: [{ title: 'main() frame', vars: 'list → @Heap#101', cls: 'f-active' }], heap: [{ title: 'ArrayList<String> @101', vars: '[empty]', cls: '' }], notes: ['T = type parameter'] }, // थप steps... ];
4
index.json मा lab link गर्नुस्:
{ "id":"j10", "name":"Generics", "lab":"generics.html", "labSection":"intro" }
Dashboard मा 🧪 lab button automatically appear हुन्छ।
05 Notes र SVG Files Link गर्ने
1
SVG file सही folder मा राख्नुस्:
# Java notes: assets/image/java/mynote.svg # DSA notes: assets/image/dsa/tree_notes.svg
2
index.json मा notes array मा path थप्नुस्:
{ "id": "j5", "name": "JVM Architecture", "notes": [ "../../assets/image/java/jvmarchitecture_drawio.svg", "../../assets/image/java/my_new_note.svg" ] }
Notes page मा automatically 📐 open button देखिन्छ।
3
Lab भित्र पनि link गर्न सकिन्छ:
<!-- lab HTML भित्र --> <a class="btn btn-sm" href="../../assets/image/java/mynote.svg" target="_blank"> 📐 View Drawing </a>
drawio / excalidraw SVG: Browser मा directly open हुन्छ — extra software चाहिँदैन। Export as SVG गर्नुस् र assets/image/[lang]/ मा राख्नुस्।
06 Topic Status Update गर्ने
तरिका 1 — Dashboard मा dropdown (recommended)
Language page मा जानुस् → topic को छेउमा dropdown → status change गर्नुस्। localStorage मा automatically save हुन्छ — browser बन्द गरे पनि रहन्छ।
तरिका 2 — index.json directly edit गर्नुस्
contents/java/index.json मा "status": "done" manually change गर्नुस्। Note: localStorage ले JSON लाई override गर्छ — localStorage clear गर्नुस् पहिले यदि conflict भयो भने।
localStorage clear गर्न: Browser DevTools → Application → Local Storage → mlj-progress delete गर्नुस्। अनि JSON को values reflect हुन्छन्।

↗ Open SVG → browser मा drawing open · Auto-populated from index.json notes arrays · नयाँ note थप्न index.json edit मात्र
Progress Page
Real-time stats from all index.json files · Per-language progress bars · Sushasan-X Roadmap tracker

10 Pro Tips
💡
Live Server use गर्नुस् — file save गर्नासाथ browser refresh। VS Code Extension: "Live Server"।
💡
global.css मात्र change गर्नुस् design को लागि — सबै pages र labs automatically update हुन्छन् किनभने सबैले ../../web-engine/css/global.css link गर्छन्।
💡
ID convention follow गर्नुस् — java: j1,j2... · js: js1,js2... · cpp: cpp1... · dsa: d1,d2... · python: py1... Unique IDs नभए localStorage conflict हुन्छ।
💡
Git commit message pattern:
git add . git commit -m "feat(java): add Generics lab" git commit -m "content(dsa): mark LinkedList as done" git commit -m "note(java): add generics_notes.svg"
💡
Lab बनाउँदा mobile test गर्नुस् — Chrome DevTools मा toggle device toolbar (Ctrl+Shift+M)। @media(max-width:700px) already global.css मा छ।
💡
topic JSON मा labSection — lab file मा multiple tabs छन् भने specific tab मा directly link गर्न सकिन्छ:
"lab": "memory-lab.html", "labSection": "string-pool" // tab id
Dashboard बाट click गर्दा directly String Pool tab मा open हुन्छ।
