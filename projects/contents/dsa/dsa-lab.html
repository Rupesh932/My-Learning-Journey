Best Case (Ω)
Minimum operations। Linear search — first element नै target: O(1)।
Average Case (θ)
Expected operations। Linear search — middle मा target: O(n/2) = O(n)।
Worst Case (O)
Maximum operations। Linear search — last element वा not found: O(n)। Big-O = worst case।
