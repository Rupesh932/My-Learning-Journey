१. Static Keyword (साझा सदस्य)

जब हामी कुनै सदस्यलाई Static बनाउँछौँ, त्यो कुनै निश्चित Object को नभएर पूरै Class को साझा सम्पत्ति बन्छ।

  • यसले Memory को बचत गर्छ किनभने यो Method Area मा एक पटक मात्र बस्छ।
  • यसलाई Object नबनाई सिधै Class Name बाट प्रयोग गर्न सकिन्छ।
  • Static Method ले केवल Static Data लाई मात्र चलाउन पाउँछ।
public class Counter { static int count = 0; // Class Level Variable Counter() { count++; // सबै Object ले एउटै 'count' चलाउँछन् } }

२. Final Keyword (अपरिवर्तनीय)

Final ले कुनै पनि कुरालाई परिवर्तन हुनबाट रोक्छ र सुरक्षा प्रदान गर्छ।

प्रयोग प्रभाव (Effect)
Final Variable यसको मान फेर्न मिल्दैन (Constant)।
Final Method यसलाई Override गर्न मिल्दैन।
Final Class यसलाई Inherit वा Extend गर्न मिल्दैन।
